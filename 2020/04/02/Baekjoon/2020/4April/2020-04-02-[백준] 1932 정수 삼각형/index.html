1. 문제

정수 삼각형

2. 코드

  • c++
  • python

3. 해결 방법

  • dp[i+1][j] = max(dp[i+1][j], sq[i+1][j] + dp[i][j])
  • dp[i+1][j+1] = max(dp[i+1][j+1], sq[i+1][j+1] + dp[i][j])
  • 이 점화식을 사용한다.
  • 계속 더하기만 하면 중간에 겹치는 부분때문에 숫자가 엄청 커진다.
  • 그래서 max를 이용해서 최대값을 넣어준다.
