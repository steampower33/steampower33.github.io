[md] md파일에 이미지 업로드 방법

2020/01/05 카테고리 md

1. github 프로젝트의 issues 클릭 ( project에서 issues를 따로 설정하지않은 ), New issue 클릭.

2. “Leave a comment” 부분에 이미지를 드래그 앤 드롭하고, 업로드해서 나온 값을 본인의 md 파일에 붙여넣기.

3. 그럼 이 페이지의 결과가 나온다.
