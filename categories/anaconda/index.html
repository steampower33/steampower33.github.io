미리 말씀드리는것

왜 이 오류가 발생했고, 오류 해결 방법으로 오류를 해결했지만, 이 방법으로 이 오류를 해결하는 방법이 옳은지는 확신하지못합니다.
저는 단순히 임시방편이라 생각하고있고, 이 글을 보시는분들도 그렇게 알아주셨으면 합니다.

1. konlpy에서 제공하는 형태소 분석기를 사용하려는데 발생하는 오류

from konlpy.tag import *

2. 해결

해결 방법 주소

위주소의 답글이 해결 방법이다.

  1. “C:\Users\ai\Anaconda3\lib\site-packages\konlpy” 주소로 이동 후 jvm.py 란 파일 찾기.
  2. init_jvm function에 folder_suffix으로 이동.
  3. 주석 처리된 부분에 JAVA sources랑 Twitter (Okt)를 보면 그 아래에 있는 값들중에 마지막에 *이 있는 부분이 있다. 그 줄을 삭제해준다. 총 2줄

  4. 그럼 오류가 사라지고 정상 작동됨.
