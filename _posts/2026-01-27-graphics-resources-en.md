This is an archive of websites and resources that have been incredibly helpful in my graphics programming studies.

It covers everything from basic theory to practical assets and tools. I plan to **continuously update** this post as I discover new valuable materials along my journey.

---

# Simulation
- [Ten Minute Physics](https://matthias-research.github.io/pages/tenMinutePhysics/index.html?utm_source=chatgpt.com) : The best site to learn physics simulation intuitively.

# PBR (Physically Based Rendering)
- [OpenPBR](https://academysoftwarefoundation.github.io/OpenPBR/) : Resources on PBR standards and implementation.

# Learn Computer Graphics
- [Scratchapixel](https://www.scratchapixel.com/) : Explains everything from basic graphics mathematics to rendering pipelines in great detail.

# Learn Vulkan
- [Khronos Vulkan Tutorial](https://docs.vulkan.org/tutorial/latest/00_Introduction.html) : The textbook for Vulkan beginners.
- [SaschaWillems Vulkan Examples](https://github.com/SaschaWillems/Vulkan) : A massive repository of Vulkan C++ example codes.

# Tools 
- [TetGen](https://codeberg.org/TetGen/TetGen/) : A quality tetrahedral mesh generator.

# Assets
- [Mixamo](https://www.mixamo.com/#/) : Get rigged characters and animations for free.
- [ambientcg](https://ambientcg.com/) : Provides high-quality PBR textures for free.
- [FREEPBR](https://freepbr.com/) : A free PBR texture site.
- [PolyHaven](https://polyhaven.com/) : Offers high-quality HDRIs, textures, and models.

# RayMarching
- [Inigo Quilez](https://iquilezles.org/) : The bible of Raymarching and SDF (Signed Distance Functions).

# Shader Programming
- [ShaderToy](https://www.shadertoy.com/) : A platform where you can write shader code and verify results instantly on the web.
