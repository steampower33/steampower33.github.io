그래픽스 프로그래밍을 공부하면서 유용했던 사이트와 자료들을 모아둔 아카이브입니다. 

기초적인 이론부터 실무에 필요한 에셋, 툴까지 다양하게 정리해 두었으며, 앞으로 공부하며 알게 되는 좋은 자료들을 **지속적으로 업데이트**할 예정입니다.

---

# Simulation
- [Ten Minute Physics](https://matthias-research.github.io/pages/tenMinutePhysics/index.html?utm_source=chatgpt.com) : 물리 시뮬레이션을 직관적으로 배울 수 있는 최고의 사이트입니다.

# PBR (Physically Based Rendering)
- [OpenPBR](https://academysoftwarefoundation.github.io/OpenPBR/) : PBR의 표준과 구현에 대한 자료입니다.

# Learn Computer Graphics
- [Scratchapixel](https://www.scratchapixel.com/) : 그래픽스의 기초 수학부터 렌더링 파이프라인까지 아주 상세하게 설명되어 있습니다.

# Learn Vulkan
- [Khronos Vulkan Tutorial](https://docs.vulkan.org/tutorial/latest/00_Introduction.html) : Vulkan 입문자의 교과서입니다.
- [SaschaWillems Vulkan Examples](https://github.com/SaschaWillems/Vulkan) : 방대한 양의 Vulkan C++ 예제 코드 저장소입니다.

# Tools 
- [TetGen](https://codeberg.org/TetGen/TetGen/) : 퀄리티 높은 사면체 메쉬(Tetrahedral Mesh) 생성 도구입니다.

# Assets
- [Mixamo](https://www.mixamo.com/#/) : 리깅된 캐릭터와 애니메이션을 무료로 구할 수 있습니다.
- [ambientcg](https://ambientcg.com/) : 고품질 PBR 텍스처를 무료로 제공합니다.
- [FREEPBR](https://freepbr.com/) : 무료 PBR 텍스처 사이트입니다.
- [PolyHaven](https://polyhaven.com/) : 고퀄리티 HDRI, 텍스처, 모델을 제공합니다.

# RayMarching
- [Inigo Quilez](https://iquilezles.org/) : 레이마칭과 SDF(Signed Distance Functions)의 바이블 같은 블로그입니다.

# Shader Programming
- [ShaderToy](https://www.shadertoy.com/) : 웹에서 쉐이더 코드를 작성하고 결과를 바로 확인할 수 있는 플랫폼입니다.
